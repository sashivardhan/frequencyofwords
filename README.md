# frequencyofwords
app can be started using "npm start" or "node index.js".
index.js has the logic to handle routing and counting the frequency of words.
'request-promise' module is used to read the remote file asyncronously.
'express' module is used to handle routing and starting server.
'body-parser' is used to read the posted inputs from client.
views/index.ejs is used to render the view
public/styles/css has custom styling.Bootstrap is used for styling tables and forms.

App is deployed on heroku:
